# MindMate AI
MindMate AI is a production-ready, enterprise-grade, AI-powered Mental Wellness Platform specifically designed for students preparing for competitive examinations (JEE, NEET, UPSC, etc.).

## Vision
To help students identify emotional stress patterns, prevent burnout, improve focus, and receive personalized AI-powered wellness guidance throughout their academic journey.

## Features
- **Daily Mood Tracking**: Track daily emotional states.
- **AI Reflection Journal**: Analyze journal entries for stress triggers using Gemini AI.
- **AI Wellness Coach**: Chat with a 24/7 AI companion for support.
- **Personalized Wellness Roadmap**: Generate custom schedules tailored to study hours and stress.
- **Focus Timer**: Pomodoro, Deep Focus, and Advanced timers.
- **Emergency Toolkit**: Quick relief grounding and breathing exercises.
- **Analytics Dashboard**: Visual representations of wellness scores and burnout risk.
- **Gamification**: Streaks, leveling up, and achievements to keep you motivated.

## Technologies Used
- HTML5, CSS3, Vanilla JavaScript (ES6 Modules)
- Bootstrap 5 for rapid layout and responsiveness
- Chart.js for beautiful analytics visualization
- AOS for scroll animations
- Google Gemini API (1.5 Flash) for all AI features
- LocalStorage for entirely local, secure, and private data retention

## Setup & Run Locally
1. Clone the repository.
2. Ensure you have a Google Gemini API Key.
3. Serve the directory using any static server (e.g., `npx serve .` or VS Code Live Server).
4. Navigate to `index.html`.
5. Provide your API Key in the "AI Journal" tab when prompted.
